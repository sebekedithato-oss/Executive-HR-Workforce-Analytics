# Executive-HR-Workforce-Analytics
1. Organizations require visibility into workforce performance, employee satisfaction, attrition risk, absenteeism, and operational workforce trends to support strategic HR decision-making.

2. This project is an Executive HR Workforce Analytics Solution built in Power BI to support workforce intelligence, employee retention analysis, operational HR reporting, and executive decision-making.

3. Objectives:

monitor workforce KPIs,
analyze attrition risk,
identify workforce engagement trends,
support employee-level investigation,
provide executive workforce insights.

4. Tools:
   Power BI - Dashboard Development
   Power Query - Data Cleaning
   DAX - KPI Calculations
   Excel/CSV - Data Source

5. Dashboard Features
**Executive Overview**
workforce KPIs,
headcount trends,
salary analysis,
attrition overview.

**Workforce Risk & Performance Analytics**
attrition heatmap,
salary vs performance analysis,
satisfaction correlation,
absenteeism analysis,
interactive tooltips.

**Employee Workforce Analysis**
employee-level investigation,
conditional formatting,
operational workforce filtering,
employee search capability.
